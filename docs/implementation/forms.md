# Formspree Integration

## Waitlist Form

- Single email input
- Action posts to Formspree endpoint
- Show confirmation state after submission
- Keep copy gentle and reassuring

## UX Requirements

- Input: translucent background, soft border
- Button: gradient, subtle lift on hover
- Focus: soft glow, no harsh outline

## Validation

- Email input should use `type="email"`
- Provide minimal error messaging in calm tone

## Security

- No backend required beyond Formspree
