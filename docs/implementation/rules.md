# Implementation Rules (Non-Negotiable)

## Visual

- Use glassmorphism with `backdrop-blur` and semi-transparent layers
- Pastel gradients only (no neon)
- Soft shadows; avoid harsh contrast
- Rounded corners throughout

## Motion

- Slow, gentle transitions only
- No aggressive or sharp animations
- Respect `prefers-reduced-motion`

## Content

- Calm, empathetic tone
- No urgency or pressure language

## Sections

- Must include all 7 sections in order
- Each section must follow the specs in [docs/implementation/sections.md](docs/implementation/sections.md)

## Accessibility

- Maintain readable contrast
- Visible, soft focus states
- Semantic structure and headings
