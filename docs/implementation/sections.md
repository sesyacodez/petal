# Section Specifications

## 1) Navbar (Glass, Sticky)

**Layout**

- Floating glass bar with rounded corners
- Left: logo, right: nav links

**Styles**

- `backdrop-blur-md`, `bg-white/50`, `border-white/20`
- Subtle shadow

**Behavior**

- Sticky on scroll
- Link hover: soft underline or glow

---

## 2) Hero

**Content**

- Headline: “A softer way to stay productive.”
- Subheadline: “petal helps you focus without pressure, guilt, or burnout.”
- Primary CTA: “Join the waitlist”

**Visuals**

- Background gradient: pink → white → blush
- Floating glass shapes/petals (slow animation)
- Optional subtle grain overlay

**Motion**

- Fade-in text on load
- Slow floating animation for decorative elements

---

## 3) Emotional Validation

**Content**

- 2–3 empathetic statements
- Example: “Most productivity apps feel loud.”

**Design**

- Glass cards with soft blur
- Gentle hover glow

---

## 4) Philosophy / How petal Helps

**Content**

- 3–4 points
- Examples: “One task at a time”, “No streaks or judgment”

**Design**

- Icon + text in glass containers
- Subtle gradient edge on hover

---

## 5) Features

**Content**

- Grid of glass cards
- Each: icon, title, 1–2 line description

**Design**

- Rounded-3xl, backdrop blur
- Soft gradient border/shadow
- Hover: tiny lift, shadow deepen

---

## 6) Soft CTA (Formspree)

**Content**

- Reassuring copy, no urgency
- Email field + CTA button

**Design**

- Glass form container
- Input with translucent background
- Button: pink gradient, soft lift

**Behavior**

- Form submits to Formspree endpoint
- Confirmation message after submission

---

## 7) Footer

**Design**

- Minimal glass bar
- Logo + small links (Privacy, Contact)
- Muted text, soft palette
